Class SyncManager

Manages synchronization between the Obsidian vault and the database. This class handles file system events and ensures proper synchronization between the file system and the database state.

Remarks

Sets up a FileSystemWatcher to monitor the Obsidian vault directory for changes to markdown files. The watcher is configured to track various file system events including changes, creations, deletions, and renames.
